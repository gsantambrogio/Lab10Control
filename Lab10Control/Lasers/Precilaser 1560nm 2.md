# Description
Precilaser used for SFG together [[Precilaser 1051]]

# Model
Precilaser FECL-SF-1560nm

# Specifications
- Output power: 25 mW
- Wavelength: 1560.48 nm
- Bandwidth: 1.01 MHz

# Locking
This laser is locked to the comb using the 1560 nm output of the comb called RP1. This output is the one used for locking the comb to the fibre link.  The beam for the locking is samplede by a fibre-beam splitter: 25% for the locking and 75% for SFG with [[Precilaser 1051]].
The PID is done by the RedPitaya1.
