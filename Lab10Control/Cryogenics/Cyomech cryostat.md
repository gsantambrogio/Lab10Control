# Description
Cryostat for the buffer-gas source. 

# Model
PT425 - CPA1114
(CPA1100 series)

# Specifications
- Electrical specifications: 380/415 VAC 50 Hz, 13.5 kW (12 kW steady state), 23 A
- System helium pressure: 18.6 ± .34 bar @ 50 Hz
- Cold head max load 1st stage: 10 kg
- Cold head max load 2st stage: 5 kg
- ![[Cryo-waterFlow.png]]

# Maintenance
